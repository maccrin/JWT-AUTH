# JWT Auth API

A backend-only JWT-based authentication service built with Node, Express and PostgreSQL.

## Features

- User registration & login  
- Password hashing (bcrypt)  
- JWT issuance & verification  
- Protected routes  
- Automated tests & CI/CD pipeline
